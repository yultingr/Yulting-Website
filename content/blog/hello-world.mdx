# Hello World

Welcome to my new personal website and blog! I built this site using Next.js, TypeScript, Tailwind CSS, and MDX.

## Why I Built This

I wanted a place to share my thoughts on software development and document my learning journey. Having a personal website gives me a space to:

- Share projects I'm working on
- Write about things I'm learning
- Connect with other developers

## Tech Stack

Here's what powers this site:

- **Next.js** with App Router for the framework
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **MDX** for blog content

I plan to write about web development, tools I find useful, and interesting problems I encounter. Stay tuned!
